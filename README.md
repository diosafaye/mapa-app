**MAPA: Bohol Heritage Asset Management and Real-time Disaster Risk Resilience App**

**About**

A web-based platform designed to streamline the reporting and verification of damage to heritage sites, ensuring timely intervention and data-driven conservation efforts. This project was developed as part of a technical initiative to leverage modern web technologies for cultural heritage preservation.

Key Features include:

1. Damage Reporting: Users can submit detailed reports of damage to heritage sites, including descriptions, damage levels, and photographic evidence.
2. Real-time Storage: Integration with Supabase Storage for secure, scalable handling of image uploads.
3. Status Tracking: Built-in status tracking to monitor reports from "Pending Review" to "Verified."
4. Alert Sytem for DRRM: Alerts users when there are disaster risks in a certain area.

**Tech Stack**
Frontend: React, Vite
Styling: Tailwind CSS, Lucide React (Icons)
Backend & Database: Supabase (PostgreSQL)
